## Current Product Summary

Dungeon Torch Crawler is a browser-playable torchlit ARPG slice. It combines authored dungeon composition, local assets, deterministic game systems, and a fixed-camera Three.js renderer. The project is currently best understood as a vertical slice rather than a finished content-complete game.
